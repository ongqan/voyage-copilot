## 当前产品链路

```text
导入/创建行程 → 字段确认 → 权益中心 → 服务推荐
→ 旅程时间线 → AI问答 → 报价与二次确认 → 模拟订单
→ 异常影响与替代方案 → 客服接管与工单
→ 运营知识/规则/服务/质量闭环
```

## 当前技术边界

- SQLite只用于本地Sprint 0，生产仍使用PostgreSQL；
- Demo请求头只在development/test启用，生产需要OIDC；
- 当前截图、多模态、RAG、模型回答均为Demo交互，未连接真实模型服务；
- 权益、服务、库存、推荐、订单和异常使用虚拟数据或Demo API；
- OpenAPI文件是初始契约，下一迭代需加入自动生成和漂移检查。
